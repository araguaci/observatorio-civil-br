Formulação Estrutural · P3 Invertido
O sistema capturado demonstra sua natureza com precisão clínica neste contraste: velocidade máxima para punição dos que ameaçam o controle narrativo e do poder; velocidade mínima para os que desviam recursos dentro da estrutura.

Os que desviam são funcionais ao sistema. Os que protestam são ameaça ao sistema.

Esta não é uma falha. É o sistema operando conforme seu design.

Conclusão estrutural
O P3 não é uma distorção do sistema — é sua função primária. Um sistema de extração precisa punir quem o ameaça e proteger quem o alimenta. A seletividade punitiva não é um bug; é a feature que garante a sobrevivência do sistema capturado.

O Sistema que se Revela a Si Mesmo
A assimetria pandêmica não é o produto de um sistema imperfeito. É o produto de um sistema que funciona exatamente como projetado.

Um sistema de extração tem dois imperativos estruturais: punir quem ameaça o controle e proteger quem alimenta o fluxo. O P3 (Seletividade Punitiva) é o mecanismo que executa ambos simultaneamente — com o mesmo aparato institucional, operando em velocidades opostas conforme o perfil do alvo.

O contraste Jan/2023 × Pandemia expõe o design com uma clareza que nenhum documento isolado conseguiria: o STF que bloqueou a investigação dos mandantes pandêmicos em junho de 2021 foi o mesmo que condenou ~1.500 manifestantes com velocidade recorde a partir de janeiro de 2023. Não é contradição. É consistência — com a lógica do sistema capturado.

A pergunta correta não é "por que o sistema falhou com os desvios pandêmicos?". A pergunta correta é: quem se beneficia quando o sistema processa em velocidades tão distintas? A resposta está no desfecho: os mandantes pandêmicos tornaram-se ministros. Os manifestantes tornaram-se presos. O sistema funcionou.
